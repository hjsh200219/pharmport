## [LIMITATION]

1. **성분명 파싱 한계**: 성분 수 계산은 콤마+괄호 규칙으로 처리하였으나,  
   일부 비정형 성분명(한글 복합표현, 세미콜론 구분 등)은 과소 계산될 수 있음.
2. **IsDeleted=false 필터**: 삭제된 코드(IsDeleted=true) 제외. 히스토리 분석은 별도 필요.
3. **비-9자리 코드 제외**: 9건의 테스트/TI 코드는 분석에서 제외함.
4. **성분명 NULL 처리**: 복합제 12,435건 중 99건은 성분명 NULL로, 분석 2에서 제외됨.
5. **코드 의미 추정**: 5-6자리 = '00' → 단일제 가정은 CLAUDE.md의 명세를 따름.  
   실제 약학적 의미는 심평원 공식 코드 체계 문서로 교차 검증 필요.
